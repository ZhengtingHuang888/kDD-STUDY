# JUSPYTER NOYEBOOK

## 简介：

Jupyter Notebook 的本质是一个 Web 应用程序，便于创建和共享文学化程序文档，支持实时代码，数学方程，可视化和 [markdown](https://baike.so.com/doc/6949586-7171987.html)。 用途包括:数据清理和转换，数值模拟，统计建模，机器学习等等 。

##基础语法（普通的python文件代码）：

新建一个python文件，在web上进行编译和运行，查看运行结果
